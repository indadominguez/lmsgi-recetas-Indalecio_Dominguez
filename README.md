# lmsgi-recetas-Indalecio_Dominguez

Desarrollo de una pequeña aplicación de visualización de recetas de cocina que le directamente un documento XML desde el navegador, utilizando los siguientes lenguajes y tecnologías:

XML para la representación de datos.
XSD (XML Schema) para la validación del documento.
JavaScript para cargar y mostrar dinámicamente los datos en una tabla HTML.
